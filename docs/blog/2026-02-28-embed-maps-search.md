MeshMonitor v3.7.3 introduces two major features: **Embeddable Map Widgets** and **Unified Message Search**.

## Embed Maps

Share a live view of your mesh network on any website with a simple `<iframe>` embed. Create **embed profiles** in Settings with channel filters, map defaults, and allowed origins. The embedded map includes hop-colored markers, rich node popups, neighbor info lines, and traceroute paths — all secured with CSP `frame-ancestors` so only your approved domains can embed it.

[Read more about Embed Maps](https://meshmonitor.org/features/embed-maps)

## Unified Message Search

Search across all your channels, DMs, and MeshCore messages in one place. Open the search modal from the sidebar or press **Ctrl+K** (Cmd+K on Mac). Filter by sender, channel, date range, and scope — then click any result to jump directly to the message with highlight animation. Results respect your channel permissions, and programmatic access is available via the REST API.

[Read more about Message Search](https://meshmonitor.org/features/message-search)
